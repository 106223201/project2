# project2

Clone project2 to your computer and run it via localhost. Check out the following features.
1. User login and register page: *login-register.php*
2. Job listings: *jobs.php*
3. Application form: *apply.php*
4. Success confirmation or Error page: *process_eoi.php*
5. User dashboard for checking application progress: *dashboard.php*
6. Manager signup: *manager-register.php*
7. Manager login: *manager-login.php*
8. Manager dashboard: *manage.php*
9. Changing nav-bar based on modes: *nav.inc*, *manager-nav.inc*, *user-nav.inc*
